# AI-Learning Platform

## Installation
```bash
poetry install
```

## Start the Server
```bash
poetry run python manage.py runserver
```

## Environment Variables Schema
```env
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=
DJANGO_SECRET_KEY=
DJANGO_ADMIN_USERNAME=
DJANGO_ADMIN_EMAIL_ID=
DJANGO_ADMIN_PASSWORD=
```
